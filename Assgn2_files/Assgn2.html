# Define a function to clean text data
def clean_text(text):
    # Convert text to lowercase
    text = text.lower()
    # Remove any non-alphabetic characters (punctuation, numbers, etc.)
    text = re.sub(r'[^a-z\s]', '', text)
    # Tokenize the text
    tokens = nltk.word_tokenize(text)
    # Remove English stopwords
    tokens = [word for word in tokens if word not in stopwords.words('english')]
    # Lemmatize tokens to reduce them to their base form
    lemmatizer = WordNetLemmatizer()
    tokens = [lemmatizer.lemmatize(word) for word in tokens]
    # Join tokens back into a cleaned string
    cleaned_text = ' '.join(tokens)
    return cleaned_text

import pandas as pd
import plotly.express as px
import requests

# 1. Load your county-level dataset
df = pd.read_csv("US_counties_COVID19_health_weather.csv")

# 2. Convert date column to datetime if you need to filter by year
df["date"] = pd.to_datetime(df["date"], errors="coerce")
df_2020 = df[df["date"].dt.year == 2020]

# 3. Sum cases by FIPS for the entire year 2020
df_2020 = df_2020.groupby("fips", as_index=False)["cases"].sum()

# 4. Make sure FIPS codes are zero-padded to 5 digits
df_2020["fips"] = df_2020["fips"].astype(str).str.zfill(5)

# 5. Get the county GeoJSON file
geojson_url = "https://raw.githubusercontent.com/plotly/datasets/master/geojson-counties-fips.json"
counties_geojson = requests.get(geojson_url).json()

# 6. Create a choropleth map
fig = px.choropleth(
    df_2020,
    geojson=counties_geojson,
    locations="fips",        # Column containing the 5-digit FIPS codes
    color="cases",           # Column to color by
    color_continuous_scale="Reds",
    scope="usa",             # Focus on the USA
    labels={"cases": "COVID-19 Cases"},
    title="Total COVID-19 Cases by County (2020)"
)

fig.update_layout(margin={"r":0,"t":40,"l":0,"b":0})
fig.show()

The choropleth map illustrates the geographic pattern of total COVID-19 cases at a county level in the United States for the year 2020. The counties are depicted in varying shades proportional to the number of cases reported throughout the year, with darker shades indicating more reported cases. The visualization enables several observations to be made:

Regional Hotspots: Darker red counties indicate where there was a higher cumulative total of COVID-19 cases in the year 2020. This is most commonly located in more densely populated regions, i.e., large cities.

Population Centers: Note that counties with or near major cities will have more cases, perhaps due to both denser population and more testing or reporting.

Geographical Distribution: The map enables you to easily observe how cases were spread across the country. Lighter-colored counties generally have less, often associated with more rural or thinly populated areas.

Variability Across States: While some states might have higher total case numbers, a county-level analysis identifies specific locations that have been significantly hit. This makes it easier to pinpoint clusters or potential epicenters of outbreaks within states. Overall, the map provides a comprehensive, detailed image of the distribution of cases of COVID-19 across the United States in 2020, indicating both overall regional patterns and local differences in case burdens.
